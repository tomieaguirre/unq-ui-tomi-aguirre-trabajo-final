# Palabras Encadenadas

Trabajo Final Integrador de Interfaces de Usuario (UNQ – 1°C 2026).

Aplicación web desarrollada con React donde el objetivo es formar la cadena más larga posible de palabras válidas antes de que se agote el tiempo.

## Demo

https://palabras-encadenas.vercel.app/

## Tecnologías

- React 19
- React Router
- Vite
- CSS Modules
- Lucide React
- LocalStorage

## Instalación

Clonar el repositorio:

```bash
git clone https://github.com/tomieaguirre/unq-ui-tomi-aguirre-trabajo-final
```

Ingresar al proyecto:

```bash
cd unq-ui-tomi-aguirre-trabajo-final
```

Instalar dependencias:

```bash
npm install
```

Iniciar el servidor de desarrollo:

```bash
npm run dev
```

Abrir en el navegador:

```
http://localhost:5173
```

## Funcionalidades

- Juego de Palabras Encadenadas.
- Validación de palabras mediante la API de la cátedra.
- Puntaje por cantidad de letras.
- Temporizador por turno.
- Ranking local (Top 10).
- Configuración de audio.
- Interfaz responsive.
